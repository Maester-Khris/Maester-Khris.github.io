Project description Text Ninja: ML Based Advanced NLP text processing tool

Text Ninja is a strategic enhancement designed to complement the custom conversational AI agent, Gemini, addressing two significant challenges: cognitive overload and overgeneralization in responses. As part of the Snappy project, Text Ninja aims to improve the interaction quality when Gemini operates in active chatbot mode during conversations. To tackle these issues, Text Ninja incorporates two main modules: Text Summarization and Text Contextualization.

  • Text Summarization: This module employs an extractive approach utilizing a multilingual pre-trained BERT neural network model.
  • Text Contextualization: Inspired by NLP text augmentation techniques, this module performs contextualized keyword data augmentation.This is achieved using a transformer-based neural network with an AutoSeqtoSeq architecture and a multilingual pre-trained BERT model for Next Word Prediction tasks.
  • Main tech stack: Python, Flask, Flasgger, Spacy, Bert LLM, AutoSeq2Seq

Project description Springboot Live Skill Voter app

We designed a web app that allows users or guest to decide which Language/Framerwork/Tools is the best by participating to a live perpetual vote. To simulate data coming from multiple sources in a real-time fashion way, we implemented a multi-threaded environment where threads produce new vote on existing resources. This project implements real-time data updates using asynchronous client data submission, leveraging Axios for robust client-server communication. Also it employs server-sent events through reactive programming and best practices for concurrency management utilizing Spring boot Reactor Sinks and Mono.fromRunnable APIs to create a an asynchronous non-blocking web-thread producer for efficient event handling.
Main tech stack: Java 8, Springboot, SpringWebFLux, Flyway, Mysql, Server Sent Event, Websocket, Docker, Github Action
