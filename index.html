Project description Springboot Live Skill Voter app

We designed a web app that allows users or guest to decide which Language/Framerwork/Tools is the best by participating to a live perpetual vote. To simulate data coming from multiple sources in a real-time fashion way, we implemented a multi-threaded environment where threads produce new vote on existing resources. This project, built with Java 8 and Spring boot Framework implements real-time data updates using asynchronous client data submission, leveraging Axios for robust client-server communication. Also it employs server-sent events through reactive programming and best practices for concurrency management utilizing Spring boot Reactor Sinks and Mono.fromRunnable APIs to create a an asynchronous non-blocking web-thread producer for efficient event handling.
